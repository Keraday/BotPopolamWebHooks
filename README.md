# Telegram Expense Split Bot

> Бот для расчёта общих расходов в группе — пока в разработке!

Этот бот помогает друзьям легко делить общие траты: кто сколько потратил, кто кому должен — всё автоматически.

**В активной разработке** 

## Основные команды 

- `/start` — приветствие и справка  
- `/add <сумма>` — добавить свою трату  
- `/calc` — рассчитать, кто кому должен  
- `/reset` — сбросить данные чата

##  Технологии

- Go (Golang)
- Telegram Bot API
- Webhooks (в планах)
